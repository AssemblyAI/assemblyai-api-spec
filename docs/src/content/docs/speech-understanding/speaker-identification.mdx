---
title: Speaker Identification
description: Identify and match speakers across audio files
---

Speaker Identification enables you to identify and match speakers across multiple audio files. This goes beyond basic diarization by providing persistent speaker identities.

## How it works

1. Submit audio with speaker labels enabled
2. The API identifies unique speakers and assigns consistent labels
3. Use the speaker embeddings for cross-file matching

## Getting started

```bash
curl https://api.assemblyai.com/v2/transcript \
  --header "Authorization: YOUR_API_KEY" \
  --header "Content-Type: application/json" \
  --data '{
    "audio_url": "https://assembly.ai/wildfires.mp3",
    "speaker_labels": true
  }'
```

See the [API Reference](/docs/api-reference) for the full parameter list.
