Entity Detection identifies and classifies entities in your transcripts â€” including people, organizations, locations, dates, and more.

## Enable entity detection

Add `entity_detection: true` to your transcription request:

```bash
curl https://api.assemblyai.com/v2/transcript \
  --header "Authorization: YOUR_API_KEY" \
  --header "Content-Type: application/json" \
  --data '{
    "audio_url": "https://assembly.ai/wildfires.mp3",
    "entity_detection": true
  }'
```

## Supported entity types

| Entity Type | Examples |
|-------------|----------|
| `person_name` | John Smith, Dr. Jones |
| `organization` | AssemblyAI, Google |
| `location` | San Francisco, California |
| `date` | January 1st, 2024 |
| `phone_number` | (555) 123-4567 |
| `email_address` | user@example.com |

See the [API Reference](/docs/api-reference) for the full response schema.
