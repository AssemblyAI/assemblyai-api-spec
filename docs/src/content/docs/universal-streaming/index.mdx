Universal Streaming delivers real-time transcription with ~300ms immutable transcripts. It's designed for voice agents, live captions, and any application that needs instant speech-to-text.

<Aside type="note">
  Streaming is available in both NA and EU regions. Use `streaming.eu.assemblyai.com` for EU data residency.
</Aside>

## Features

| Feature | Description |
|---------|-------------|
| [Multilingual](/docs/universal-streaming/multilingual) | Transcribe in English, Spanish, French, German, Italian, and Portuguese |
| [Turn detection](/docs/universal-streaming/turn-detection) | Intelligent endpointing for natural turn-taking |
| [Keyterms prompting](/docs/universal-streaming/keyterms-prompting) | Boost accuracy for up to 100 specific terms |
| [Temporary tokens](/docs/universal-streaming/temporary-token) | Authenticate front-end clients securely |
