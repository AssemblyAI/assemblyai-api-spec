## Overview

This guide walks you through transcribing your first audio file with AssemblyAI. You will learn how to submit an audio file for transcription and retrieve the results using the AssemblyAI API.

When transcribing an audio file, there are three main things you will want to specify:

1. The speech models you would like to use (required).
2. The region you would like to use (optional).
3. Other models you would like to use like Speaker Diarization or PII Redaction (optional).

## Step 1: Set up your API credentials

First, configure your API endpoint and authentication:

<Tabs>
<TabItem label="Python SDK">

```python
import assemblyai as aai

aai.settings.api_key = "YOUR_API_KEY"
```

</TabItem>
<TabItem label="Python">

```python
import requests

base_url = "https://api.assemblyai.com"
headers = {"authorization": "YOUR_API_KEY"}
```

</TabItem>
<TabItem label="JavaScript SDK">

```javascript
import { AssemblyAI } from "assemblyai";

const client = new AssemblyAI({ apiKey: "YOUR_API_KEY" });
```

</TabItem>
<TabItem label="JavaScript">

```javascript
const axios = require("axios");

const baseUrl = "https://api.assemblyai.com";
const headers = { authorization: "YOUR_API_KEY" };
```

</TabItem>
</Tabs>

## Step 2: Submit a file for transcription

<Tabs>
<TabItem label="Python SDK">

```python
audio_url = "https://assembly.ai/wildfires.mp3"

config = aai.TranscriptionConfig(
    speech_model=aai.SpeechModel.best,
)

transcript = aai.Transcriber().transcribe(audio_url, config=config)
```

</TabItem>
<TabItem label="JavaScript SDK">

```javascript
const audioUrl = "https://assembly.ai/wildfires.mp3";

const transcript = await client.transcripts.transcribe({
  audio_url: audioUrl,
  speech_model: "best",
});
```

</TabItem>
</Tabs>

## Step 3: Get the transcription result

<Tabs>
<TabItem label="Python SDK">

```python
if transcript.status == aai.TranscriptStatus.error:
    raise RuntimeError(f"Transcription failed: {transcript.error}")

print(transcript.text)
```

</TabItem>
<TabItem label="JavaScript SDK">

```javascript
if (transcript.status === "error") {
  throw new Error(`Transcription failed: ${transcript.error}`);
}

console.log(transcript.text);
```

</TabItem>
</Tabs>

<Aside type="tip">
  You can also upload local files. Pass a local file path instead of a URL and the SDK will handle the upload automatically.
</Aside>
