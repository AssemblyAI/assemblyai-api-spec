AssemblyAI offers multiple speech models optimized for different use cases. Select the model that best fits your accuracy, latency, and language requirements.

<Aside type="note">
  Universal-3-Pro is recommended for most use cases requiring the highest accuracy.
</Aside>

## Available models

| Model | ID | Languages | Best for |
|-------|-----|-----------|----------|
| Universal-3-Pro | `universal-3-pro` | 6 | Highest accuracy, prompting |
| Universal-2 (Best) | `best` | 99 | Broad language support |
| Universal-2 (Nano) | `nano` | 99 | Low-cost, high-volume |

## Setting the speech model

Pass the `speech_model` parameter when creating a transcript:

```bash
curl https://api.assemblyai.com/v2/transcript \
  --header "Authorization: YOUR_API_KEY" \
  --header "Content-Type: application/json" \
  --data '{
    "audio_url": "https://assembly.ai/wildfires.mp3",
    "speech_model": "universal-3-pro"
  }'
```
