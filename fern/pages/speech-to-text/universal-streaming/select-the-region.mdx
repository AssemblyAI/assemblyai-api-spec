---
title: "Select the Region"
---

The default region for Streaming STT is US, with base URL `streaming.assemblyai.com`. To use our EU server, change the base URL to `streaming.eu.assemblyai.com`.

| Region       | Base URL                      |
| ------------ | ----------------------------- |
| US (default) | `streaming.assemblyai.com`    |
| EU           | `streaming.eu.assemblyai.com` |

<br />

To use the EU endpoint for streaming, set the base URL to `streaming.eu.assemblyai.com`.
