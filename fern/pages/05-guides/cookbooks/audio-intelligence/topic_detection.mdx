In this guide we will show you how to identify different topics in the trancript using the [Topic Detection model](https://www.assemblyai.com/docs/audio-intelligence/topic-detection). This is a fine-tuned model trained to detect over 700 topics/sub-topics from the [IAB taxonomy](https://www.iab.com/guidelines/content-taxonomy/).

## Quickstart

```python
import assemblyai as aai

aai.settings.api_key = "API_KEY"
transcriber = aai.Transcriber()
audio_url = ("YOUR_AUDIO_URL")

config = aai.TranscriptionConfig(iab_categories=True)
transcript = transcriber.transcribe(audio_url, config)

for result in transcript.iab_categories.results:
    print(result.text)
    print(f"Timestamp: {result.timestamp.start} - {result.timestamp.end}")
    for label in result.labels:
        print(f"{label.label} ({label.relevance})")

for topic, relevance in transcript.iab_categories.summary.items():
    if relevance > 0.5:
        print(f"Audio is {relevance * 100}% relevant to {topic}")
```

## Get Started

Before we begin, make sure you have an AssemblyAI account and an API key. You can [sign up for an AssemblyAI account](https://www.assemblyai.com/app) and get your API key from your dashboard.

## Step-by-Step Instructions

Install the SDK:


```bash
pip install assemblyai
```

Import the SDK and set your AssemblyAI API key.


```python
import assemblyai as aai

aai.settings.api_key = "API_KEY"
```

Use AssemblyAI to transcribe a file and save the transcript. Make sure to set the parameter `iab_categories` to `True`.


```python
transcriber = aai.Transcriber()

audio_url = ("YOUR_AUDIO_URL")

config = aai.TranscriptionConfig(iab_categories=True)

transcript = transcriber.transcribe(audio_url, config)
```

Use the returned transcript response to print out the parts of the transcript that were tagged with topics, their timestamps, and the topic label and relevance.


```python
for result in transcript.iab_categories.results:
    print(result.text)
    print(f"Timestamp: {result.timestamp.start} - {result.timestamp.end}")
    for label in result.labels:
        print(f"{label.label} ({label.relevance})")
```

In addition, you can print out a summary of all the topics in the transcript.


```python
for topic, relevance in transcript.iab_categories.summary.items():
    if relevance > 0.5:
        print(f"Audio is {relevance * 100}% relevant to {topic}")
```
