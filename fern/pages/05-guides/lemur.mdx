Apply Large Language Models to spoken data. A Large Language Model (LLM) is a machine learning model that uses natural language processing (NLP) to generate text. LLM Gateway is a framework that lets you apply LLMs to audio transcripts, for example to ask questions about a call, or to summarize a meeting.
