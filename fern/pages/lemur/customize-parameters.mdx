## Change the model type

LeMUR features the following LLMs:

- Claude 4 Sonnet
- Claude 4 Opus
- Claude 3.7 Sonnet (_Legacy - Sunsetting on 10/3/25_)
- Claude 3.5 Sonnet (_Legacy - Sunsetting on 10/3/25_)
- Claude 3.5 Haiku
- Claude 3 Opus
- Claude 3 Haiku

<Tip>
For best results, we recommend using the latest version of your chosen model. i.e. If you wanted to use Claude Sonnet, you should use Claude 4 Sonnet rather than Claude 3.5 Sonnet.

New models are priced the same as previous models so there is no additional cost to use the latest models.

</Tip>

You can switch the model by specifying the `final_model` parameter.

Higher values result in answers that are more creative, lower values are more conservative.
