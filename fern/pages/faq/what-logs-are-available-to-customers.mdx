# What logs are available to customers?

Available logs include system, application, network, and audit logs for access, download, and deletion.

The following logs can be provided to Enterprise customers upon request:

- Header Keys
- Customer header keys included on API request
- Customer request headers included on API request
- Host information
- Customer specified API endpoint (based on model and region)
- Customer specified API endpoint method
- Customer specific API request full path

> To request for logs, please reach out to your Customer Success Manager.
