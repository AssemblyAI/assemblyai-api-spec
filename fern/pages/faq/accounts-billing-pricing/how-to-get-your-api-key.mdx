---
title: "How to an get AssemblyAI API Key"
---

To obtain your API key for AssemblyAI, follow these simple steps:

1. Visit the [AssemblyAI dashboard](https://www.assemblyai.com/app).

2. Log into your account. If you don't have an account yet, you'll need to [sign up](https://www.assemblyai.com/dashboard/signup) first.

3. Once logged in, click on "API Keys" in the left sidebar to navigate to the [API Keys page](https://www.assemblyai.com/app/api-keys).

4. Your API key will be displayed here. You can copy it using the copy button next to the key.

<Note>
  API keys may also be referred to as "authentication keys" in some contexts.
  They are the same thing — both are used to authenticate your requests to the
  AssemblyAI API.
</Note>

## Notes

- Keep your API key secure and do not share it publicly.
- You can create and manage multiple API keys from the API Keys page. See [Account Management](/docs/concepts/account-management) for more details.
- Your API key is required to authenticate your requests when using the AssemblyAI API.
