---
title: "How to get an AssemblyAI API Key"
---

To obtain your API key for AssemblyAI, follow these simple steps:

1. Visit the [AssemblyAI login page](https://www.assemblyai.com/dashboard/login).

2. Log into your account. If you don't have an account yet, you'll need to sign up first.

3. Once logged in, navigate to the [API Keys page](https://www.assemblyai.com/app/api-keys).

4. Your API key will be displayed here. API keys are listed under your project. You may also see these referred to as "authentication keys" — they are the same thing.

</Frame>

## Notes

- Keep your API key secure and do not share it publicly.
- If you need to create or delete API keys, you can do so from the same [API Keys page](https://www.assemblyai.com/app/api-keys) in the dashboard.
- Your API key is required to authenticate your requests when using the AssemblyAI API.
