---
title: "How can I use Universal-2?"
---

AssemblyAI offers several state-of-the-art speech recognition models, each optimized for different use cases.

<Note>
  For a detailed comparison of all available models, see the [Models
  overview](/docs/getting-started/models).
</Note>

There are no default models for transcription. You must specify which model or models to use via the `speech_models` parameter when making a request to the `/v2/transcript` endpoint. You can provide multiple models in priority order, and the system will automatically route to the best available model based on your request.

To learn how to select and configure speech models, see [Select the speech model](/docs/pre-recorded-audio/select-the-speech-model).

<Note title="Try Universal-3 Pro">
  Universal-3 Pro is our most powerful model yet, offering the highest accuracy
  with fine-tuning support and customization via prompting. Learn more about
  [Universal-3 Pro](/docs/pre-recorded-audio/universal-3-pro).
</Note>

Our Universal-2 model is best for out-of-the-box transcription of pre-recorded audio when you need multi-language support, high accuracy, and fast turnaround times.
